Anaku

College Canteen Food Ordering Website
Overview:
Developed a user-friendly food ordering website tailored for a college canteen to streamline meal pre-orders and minimize waiting times during peak hours. The application enhances convenience for students and operational efficiency for staff.

Tech Stack:
  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Deployment: Netlify
Features:
  • User Registration & Login: Secure authentication system allowing users to create and manage their accounts.
  • Meal Ordering System: Interactive interface for browsing menu items, placing pre-orders, and checking order status.
  • Admin Panel: Basic dashboard for managing menu items and viewing order history.
  • Real-Time Updates: Notifications for order status changes (if applicable).

Astrotinker Bot

Overview:
Developed a RISC-V CPU based robot designed to solve errors in spacecraft during the Eyantra competition. The project focused on enhancing the robot's navigation and maneuverability in space environments.

Tech Stack:
  • Hardware: RISC-V CPU
  • Algorithms: Dijkstra's Algorithm
  • Navigation: Optimized Path Planning
  • Competition: Eyantra
Features:
  • RISC-V CPU Integration: Utilized RISC-V architecture for efficient and reliable robot control.
  • Pathfinding Algorithm: Implemented Dijkstra's algorithm for accurate and efficient pathfinding.
  • Enhanced Navigation: Optimized route planning to improve navigation efficiency and maneuverability.
  • Error Resolution: Developed solutions to address and correct spacecraft errors effectively.
